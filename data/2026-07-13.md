<details>
  <summary>Details</summary>
Motivation: Kozen [1996] 提出在 Kleene 代数中引入交换性条件（形如 ab = ba 的等式）以捕获动作重排或独立性带来的等价关系。一个长期悬而未决的核心问题是：对于哪些交换性条件 C，KA+C 的等式理论是可判定的？

Method: 本文从正反两方面进行论证：负向结果证明当 C 非传递时，KA+C 的普遍性 (universality) 问题已经不可判定，进而推导出等式理论不可判定；正向结果则证明对于传递的 C，KA*+C 与 KA+C 的等式理论相同，而 KA*+C 的可判定性已知，从而得出可判定结论。

On the negative side, we show that when C is not transitive, the universality problem for KA+C is already undecidable. ### [6] [Elusive but Coverable: The Recursion-Theoretic Structure of Complete Abstract Interpretations](https://arxiv.org/abs/2607.09128)
*Nicklas Carpenter,Roberto Giacobazzi*

Main category: cs.LO

TL;DR: 本文从递归论视角研究抽象解释的局部完备性与不完备性。局部完备性弱化了全局完备性，捕捉了特定前置条件下精度无损的分析行为。作者刻画了静态与动态程序分析的区别，证明对于非平凡抽象域，诱导局部完备谓词变换子的程序类是递归论意义上的" elusive"集合（productive集，不可计算枚举），其补集同样不可枚举。这些类分别属于Π⁰₂和Σ⁰₂复杂度。有趣的是，这些局部完备程序类存在可判定的覆盖，可通过程序变换构造代表性子集的有效枚举，从外部捕获了无法从内部枚举的类。


<details>
  <summary>Details</summary>
Motivation: 传统抽象解释的全局完备性要求过于严格，实际中难以满足。作者希望从递归论角度理解局部完备性的理论边界，揭示程序的哪些类能在非平凡抽象域下实现精确分析，并探索这些类的可计算性特征。

This makes it possible to construct, via program transformation, an effective enumeration of a representative subset of programs that entirely covers this class -- capturing from the outside a class that eludes enumeration from within.
